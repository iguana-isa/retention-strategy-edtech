# План повышения Retention EdTech платформы 
Разработка стратегии повышения Retention Rate онлайн-платформы IT Resume на основе анализа активности пользователей (SQL, Metabase).

| Гипотеза | Результат |
| -------- | --------- |
| Чем быстрее пользователь входит после регистрации, тем выше Retention | Не подтвердилась (практически все входили в день регистрации) |
| Чем больше попыток отправки кода, тем выше удержание | Подтвердилась |
| Чем больше успешных решений задач, тем выше удержание | Подтвердилась |
| Положительный баланс CodeCoins (внутренняя валюта) способствует удержанию | Не подтвердилась |
