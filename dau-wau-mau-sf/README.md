

## Sticky Factor за весь доступный период
Показывает, какая доля месячной аудитории использует платформу каждый день/неделю.

Sticky Factor (DAU/MAU) = 24 / 473 * 100 = 5,07

Sticky Factor (WAU/MAU) = 119 / 473 * 100 = 25,26

Выводы:
Регулярность использования платформы низкая, это говорит о том, что большинство пользователей заходят разово и у них не формируется устойчивая регулярная активность. Вероятно, отсутсвует ежедневная мотивация к возвращению пользователей.
