# Sticky Factor за весь период
Показывает, какая доля месячной аудитории использует платформу каждый день/неделю.

**Каждый день**:
Sticky Factor = DAU/MAU * 100 = 24 / 473 * 100 = **5,07**

**Каждую неделю**:
Sticky Factor = WAU/MAU * 100 = 119 / 473 * 100 = **25,26**

### Выводы:
Регулярность использования платформы низкая, это говорит о том, что большинство пользователей заходят разово и у них не формируется устойчивая регулярная активность. Вероятно, отсутствует ежедневная мотивация к возвращению пользователей.
